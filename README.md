📌 Project Overview
A responsive web application that allows users to:
• Create tasks with titles, dates, and priorities
• Organize tasks by status (Active/Completed)
• Persist data using browser localStorage
• Manage tasks through intuitive UI
🚀 Quick Start
Basic Setup
# Clone repository
git clone https://github.com/yourusername/todo-list.git

# Open project
cd todo-list
open index.html  # Mac
start index.html # Windows

// Recommended VS Code extensions:

🛠️ Technologies
Core Stack
- Frontend: HTML5, CSS3, JavaScript (ES6)
- Storage: localStorage API
